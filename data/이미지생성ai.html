나노바나나 (Nanobanana)

🍌
나노바나나
nanobanana.ai · 한국어 특화 AI 이미지 생성
무료 시작 가능
🇰🇷
한국어 프롬프트
영어 없이 한국어로 바로 입력. 자동으로 최적 영문 프롬프트로 변환하여 생성.
🎭
다양한 스타일
사진 실사, 애니메이션, 일러스트, 수채화, 유화 등 수십 가지 스타일 사전 설정.
⚡
빠른 생성
10-30초 내 고품질 이미지 4장 동시 생성. 마음에 드는 것 선택 후 변형 가능.
✏️
인페인팅 편집
생성된 이미지의 특정 영역만 선택하여 수정. 배경 교체, 객체 제거·추가 가능.
🚀 나노바나나 시작하기
1
nanobanana.ai 접속 → 카카오/구글 간편 로그인 무료 크레딧 제공. 매일 일정량 충전되어 부담 없이 시작.
2
스타일 선택 → 한국어로 원하는 이미지 설명 입력 예: "따뜻한 카페에 앉아 책을 읽는 여성, 빈티지 느낌, 따뜻한 조명"
3
생성된 4장 중 선택 → 고해상도 다운로드 마음에 들지 않으면 "변형" 버튼으로 유사 이미지 재생성.
📸 사진 실사 🎌 애니메이션 🎨 수채화 🖌️ 유화 ✏️ 스케치 💎 3D 렌더링

ChatGPT Images 2.0 (gpt-image-2)
전 ChatGPT 플랜 · API 5월 초 오픈 · DALL-E 2·3 → 2026.05.12 종료
Image Arena 1위
🧠
Thinking Mode (Plus↑)
웹 검색 후 레이아웃 추론 → 이미지 생성 → 자동 검증·수정. 한 번에 최대 8장 일관된 배치 생성. 복잡한 씬 성공률 대폭 향상.
🔤
한국어·비라틴 문자 정확 렌더링
한국어·일본어·중국어·힌디어 텍스트를 이미지 안에 정확하게 삽입. 포스터·배너·인포그래픽 제작 수준으로 향상.
📐
2K 해상도 · 다양한 비율
최대 2K(2048px) 해상도 지원. 3:1~1:3 종횡비 자유롭게 설정. 한 프롬프트에서 8장 일관 배치 생성.
💬
대화형 반복 수정
"배경을 파란 하늘로 바꿔줘" 등 자연어로 정밀 수정. 참조 이미지 스타일 유지 변형. 이전 맥락 기억하며 일관성 유지.
⚠️ DALL-E 2·3 종료 예정: 2026년 5월 12일에 DALL-E 2·3 API가 완전 종료됩니다. 기존 코드를 gpt-image-2로 마이그레이션이 필요합니다.

이미지 프롬프트 작성법

✍️ 황금 공식: 주제 + 스타일 + 조명 + 구도 + 품질
공식 적용 예시
[주제] 노트북으로 작업하는 한국인 여성
[환경] + 모던한 카페, 큰 창문 옆
[스타일] + 사진 실사 스타일, 자연스러운 느낌
[조명] + 따뜻한 자연광, 창문에서 들어오는 빛
[구도] + 측면 미디엄 샷
[품질] + 8K, bokeh 효과, 상업 사진 품질
