Grabit (그랩잇)

웹페이지의 수많은 이미지를 클릭 한 번으로
일괄 수집하는 크롬 익스텐션

1클릭전체 이미지 저장
필터크기·형식 선택
무료기본 기능
1

어떤 문제를 해결하나?

😫 Grabit 없이
이미지 우클릭 → 다른 이름으로 저장
반복 × 수십 장
폴더 정리 필요
UI 레퍼런스 30장 = 30분 작업
✅ Grabit 사용 후
익스텐션 아이콘 클릭
크기 필터 설정 (예: 200px 이상)
"전체 다운로드" 클릭
30장 = 10초 완료
2

주요 기능

🖼️
페이지 전체 이미지 수집
현재 탭의 모든 이미지를 자동 감지. 보이지 않는 영역까지 스캔.
🔍
크기 필터
최소 가로·세로 픽셀 설정. 아이콘·로고 같은 소형 이미지 제외 가능.
📁
형식 필터
JPG·PNG·WebP·SVG 등 원하는 형식만 선택 다운로드.
👁️
미리보기 선택
다운로드 전 이미지 목록 미리보기. 필요한 것만 체크 후 일괄 저장.
3

설치 & 사용법

1
Chrome 웹스토어에서 "Grabify Image Downloader" 또는 "Grabit" 검색 후 설치유사 이름의 익스텐션 다수 존재. 별점·리뷰 확인 후 선택.
2
이미지가 많은 웹페이지로 이동 (Pinterest, Behance, 쇼핑몰 등)디자인 레퍼런스, 상품 이미지, 포트폴리오 사이트에서 특히 유용.
3
툴바에서 Grabit 아이콘 클릭 → 이미지 목록 자동 로드크기 필터로 200×200px 이상만 표시 설정 권장.
4
원하는 이미지 체크 또는 전체 선택 → 다운로드ZIP 파일로 일괄 다운로드. 파일명 자동 생성.
💡 UI 레퍼런스 수집 팁: Pinterest·Dribbble·Behance에서 마음에 드는 디자인 페이지를 열고 Grabit 실행 → 300px 이상 필터 적용 → 전체 레퍼런스 이미지 일괄 저장. Figma 프로젝트 시작 전 무드보드 구성에 최적.
