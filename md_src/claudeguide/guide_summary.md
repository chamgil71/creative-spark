# 5. [그룹 4] 자동화 & 연동 & 개발 환경

여러 서비스 앱을 유기적으로 엮어 수동으로 데이터를 복사하고 옮기는 작업을 자동화하고, 작업 환경의 효율을 끌어올리는 도구들입니다.

::: step-list
- title: "n8n (업무 자동화 워크플로우)"
  desc: "서로 다른 웹 서비스(Gmail, Slack, Notion 등)를 시각적으로 연결해, 새로운 메일이 오면 내용을 요약해 Slack으로 전송하는 식의 무인 자동 업무 루프를 형성합니다."
- title: "Playwright (브라우저 자동화 및 테스트)"
  desc: "마치 사람이 직접 사이트를 방문하듯 크롬 브라우저를 가동하여 매일 아침 특정 사이트의 정보를 수집(스크래핑)하거나 기능이 마비되지 않았는지 자동 점검합니다."
- title: "GitHub & VS Code (버전 제어 및 편집기)"
  desc: "팀원들과 코드를 조율하여 저장하는 클라우드 저장소 GitHub와, 다양한 AI 어시스턴트 플러그인을 장착해 코딩 속도를 높여주는 에디터인 VS Code입니다."
- title: "Devtools & Terminal AI (브라우저 진단 및 CLI 보조)"
  desc: "크롬 개발자 도구를 통해 웹 디자인 레이아웃을 현장에서 실시간 수정 진단하고, 터미널 명령이 꼬였을 때 AI의 도움을 받아 즉시 수정하는 도구 모음입니다."
:::

# 8. 실무 시너지 워크플로우 시나리오 (기획 & 리서치)

::: step-list
- title: "1단계: Perplexity를 통한 실시간 트렌드 분석"
  desc: "신규 사업 기획을 위해 Perplexity에 최신 해외 동향과 경쟁사 데이터를 검색하고, 신뢰할 수 있는 수치 링크와 요약본을 일차 수집합니다."
- title: "2단계: Notion에 원시 데이터 수집 및 스토리라인 설계"
  desc: "수집된 텍스트와 링크를 Notion 데이터베이스에 아카이빙하고, 마인드맵 형태로 기획서의 핵심 스토리 라인과 뼈대를 구축합니다."

  desc: "Notion에 저장된 기획 초안 문서를 NotebookLM에 업로드한 뒤, 이를 바탕으로 'Deep Dive 오디오 브리핑(대담 녹음)' 파일을 생성해 이동 중에 청취하며 추가 개선 아이디어를 도출합니다."

# 10. 실무 시너지 워크플로우 시나리오 (업무 자동화 루틴)

매일 아침 발생하는 반복 수동 작업을 지우고, 데이터 분석부터 결과 메일 발송까지 무인으로 실행하는 스마트 오피스 시나리오입니다.

::: step-list
- title: "1단계: Playwright 웹 크롤러 가동 (아침 08:00)"
  desc: "Playwright 스크립트가 매일 아침 특정 사이트들을 돌아다니며 환율 정보나 원자재 시세 데이터를 자동으로 긁어옵니다."
- title: "2단계: n8n 워크플로우 트리거 및 데이터 정제"
  desc: "수집이 완료되면 n8n이 동작하여 정제되지 않은 데이터 덩어리를 Notion 데이터베이스 테이블에 안전하게 정렬해 삽입합니다."
- title: "3단계: Claude API 연동 일일 동향 보고서 작성"
