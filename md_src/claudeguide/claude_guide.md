# 1. Claude 플랫폼의 이해 및 실무 프롬프팅

본 장에서는 Claude의 기본 구동 환경을 파악하고, 강력한 개인화 기능인 시스템 프롬프트(지침)와 SKILL.md 스킬 파일, 그리고 아티팩트(Artifacts)를 200% 활용하는 실무 노하우를 제공합니다.

## 클로드 아티팩트(Artifacts) 200% 활용법
아티팩트는 대화 흐름을 방해하지 않고 독립 패널에서 고정된 시각 결과물을 조율하는 강력한 렌더링 샌드박스입니다.

::: step-list
- title: "아티팩트 동작 원리"
  desc: "Claude가 한 세션 내에서 HTML, CSS, JS, SVG, Mermaid 다이어그램 등의 구조적 코드를 작성하면 대화창 우측에 전용 탭이 열리며, 로컬에 저장되거나 실시간 브라우저 실행이 가능해집니다."
- title: "실시간 비주얼 목업 및 시각 피드백"
  desc: "비개발자 사용자도 '우리 아이 학예회 초대장 페이지를 만들어줘'라고 지시하여 아티팩트 창에 뜬 예쁜 핑크색 웹 초대장을 직접 보면서 '바탕에 노란색 별 가루 애니메이션을 추가하고 글씨를 더 키워줘' 같이 시각적 피드백을 전달할 수 있습니다."
- title: "직관적인 결과물 소장 및 공유"
  desc: "아티팩트 패널 하단의 'Download' 버튼을 누르면 단 한 장의 HTML 파일로 저장되므로, 이 파일을 카카오톡이나 이메일로 친구들에게 전송하여 브라우저에서 바로 열어볼 수 있도록 전달할 수 있습니다."

## 클로드 루틴(Claude Routine) 가이드
에이전트가 로컬 환경에서 작업을 안전하게 수행하도록 통제하고 빌드/테스트를 유기적으로 주도하는 표준 개발 루틴 가이드라인입니다.

::: step-list
- title: "1단계: 파일 탐색 및 구조 분석 루틴"
  desc: "먼저 전체 디렉토리 트리나 `package.json`, `tsconfig.json` 등을 탐색하여 개발 언어와 프레임워크 스택을 파악한 뒤 타겟 소스 파일을 엽니다."
- title: "2단계: 정밀 부분 패치(Diff Patch) 루틴"
  desc: "전체 파일을 다시 쓰지 않고, 수정이 필요한 줄(Lines)만 정확히 포인팅하여 오리지널-업데이트 블록 구성을 통해 코드 일부만 안전하게 갈아끼웁니다."
- title: "3단계: 빌드 및 유닛 테스트 루틴"
  desc: "수정된 소스가 컴파일에 실패하지 않는지 `npm run build`나 `vitest` 등의 유닛 테스트를 CLI 쉘 상에서 스스로 가동하여 통과 유무를 검증합니다."
- title: "4단계: 자율 디버깅 무한 루프 제어"
  desc: "테스트나 컴파일 중 에러 로그가 터미널에 수신되면 에이전트가 원인을 스스로 추적하여 다시 패치하고 재검증하는 피드백 루프를 완수합니다."
:::

## MCP(Model Context Protocol) 연동 실전 예시
Claude Code에 MCP 서버를 추가 등록하여 로컬 데이터베이스를 조회하거나 실시간 웹 정보를 AI 컨텍스트에 주입하는 실제 설정 흐름입니다.

::: editor-box
- title: "MCP 서버 등록 및 사용 실전 예시"
  tag: "bash & sql"
  desc: |
    # 1. SQLite 데이터베이스 제어용 MCP 서버 등록하기 (줄번호 글자 크기 이슈 방지를 위해 백틱 코드 사용)
    npx -y @modelcontextprotocol/server-sqlite --db ./data/prod.db
    
    # 2. 쉘 내에서 데이터 질의 지시하기
    › prod.db 파일의 users 테이블에서 최근 7일 이내 회원 목록을 조회해서 이메일 도메인별 통계를 표로 그려줘.
    
    # 3. 브레이브 서치(Brave Search) 공식 웹 검색 MCP 추가하여 최신 뉴스 수집하기 (Brave API 키 등록 필요)
    npx -y @modelcontextprotocol/server-brave-search
    
    › "2026년 하반기 API 시장 동향"을 웹 검색해서 리포트를 완성해줘.

## Claude Design: UI/UX 목업 및 컴포넌트 마크업 설계
Claude의 시각 결과물 도출 능력과 SVG 인라인 코딩 스킬을 연계하여 현대적인 반응형 디자인 시스템을 단시간에 설계하는 방법입니다.

::: step-list
- title: "사용자 친화적 가계부 플래너 화면 설계"
  desc: "Claude에게 HSL 컬러 스키마가 적용된 반응형 일일 가계부 지출 차트 웹 화면을 아티팩트 패널에 렌더링하게 하여, 터치 한 번으로 카테고리별 지출 금액이 갱신되는 정밀 목업을 완성합니다."
- title: "SVG 인라인 벡터 디자인 고속 생성"
  desc: "외부 이미지 서버에서 가져올 필요 없이 '모던 카페 로고로 쓸 수 있는 미니멀한 머그잔 모양의 SVG 코드를 작성해줘'라고 지시하여 크기를 조절해도 화질이 깨지지 않는 청결한 벡터 아이콘 파일을 획득합니다."
- title: "디자인 컴포넌트 이식 및 협업 전달"
  desc: "완성된 화면의 미려한 CSS 스타일 가이드와 디자인 토큰을 복사하여, 내부 기획 디자이너 및 프론트엔드 담당자에게 전달함으로써 초기 레이아웃 기획 공유 비용을 최소화합니다."
:::
