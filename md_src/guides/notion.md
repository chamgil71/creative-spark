# 3. Notion AI — 내장 AI 어시스턴트

# 4. 필수 단축키
노션을 빠르게 쓰기 위한 핵심 단축키 모음입니다.

::: columns-grid
- title: "📝 편집 단축키"
  desc: |
    * **굵게**: `Ctrl + B`
    * **기울임**: `Ctrl + I`
    * **취소선**: `Ctrl + Shift + S`
    * **인라인 코드**: `Ctrl + E`
    * **실행취소/재실행**: `Ctrl + Z / Y`
- title: "🧭 내비게이션"
  desc: |
    * **빠른 검색**: `Ctrl + P`
    * **새 페이지**: `Ctrl + N`
    * **사이드바 토글**: `Ctrl + \`
    * **앞/뒤로 이동**: `Alt + ← / →`
    * **전체화면**: `Ctrl + Shift + F`
- title: "⚡ 블록 명령"
  desc: |
    * **블록 메뉴 열기**: `/`
    * **AI 메뉴 열기**: `Space`
    * **블록 이동**: `Ctrl + Shift + ↑↓`
    * **블록 선택**: `Esc`
    * **들여쓰기**: `Tab`
:::

# 5. 실전 노션 활용 팁
노션을 더 잘 쓰기 위한 노하우를 정리했습니다.

::: alert-box tip
- title: "🏗️ 템플릿 갤러리 활용"
  desc: "노션 공식 템플릿 갤러리에서 OKR, 독서 목록, 콘텐츠 캘린더, 버그 트래커 등 수백 가지 무료 템플릿을 바로 사용할 수 있습니다."
:::

::: alert-box tip
- title: "🔁 관계형 DB (Relation & Rollup)"
  desc: "두 데이터베이스를 연결하고 집계값을 롤업으로 가져올 수 있습니다. 예: 프로젝트 DB ↔ 작업 DB를 연결하여 프로젝트별 완료율을 자동 계산."
:::

::: alert-box tip
- title: "🌐 노션 커넥터"
  desc: "GitHub, Jira, Slack, Google Drive 등 외부 서비스를 노션과 연동할 수 있습니다. 설정 → 커넥터에서 연결하세요."
:::

::: alert-box tip
- title: "📤 공유 & 웹 퍼블리시"
  desc: "모든 페이지를 URL로 공유하거나 웹사이트로 공개할 수 있습니다. 포트폴리오, 이력서, 팀 위키로 활용됩니다."
:::
