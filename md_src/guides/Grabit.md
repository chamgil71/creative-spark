# 설치 & 사용법

::: step-list
- title: "1Chrome 웹스토어에서 \"Grabify Image Downloader\" 또는 \"Grabit\" 검색 후 설치유사 이름의 익스텐션 다수 존재. 별점·리뷰 확인 후 선택."
- title: "2이미지가 많은 웹페이지로 이동 (Pinterest, Behance, 쇼핑몰 등)디자인 레퍼런스, 상품 이미지, 포트폴리오 사이트에서 특히 유용."
- title: "3툴바에서 Grabit 아이콘 클릭 → 이미지 목록 자동 로드크기 필터로 200×200px 이상만 표시 설정 권장."
- title: "4원하는 이미지 체크 또는 전체 선택 → 다운로드ZIP 파일로 일괄 다운로드. 파일명 자동 생성."
:::

::: alert-box tip
- title: "💡 UI 레퍼런스 수집 팁:"
  desc: |
    Pinterest·Dribbble·Behance에서 마음에 드는 디자인 페이지를 열고 Grabit 실행 → 300px 이상 필터 적용 → 전체 레퍼런스 이미지 일괄 저장. Figma 프로젝트 시작 전 무드보드 구성에 최적.
:::
