# 오디오 개요 — AI 팟캐스트

문서를 두 AI 진행자가 나누는 자연스러운 팟캐스트로 변환합니다.

### 🎙️ 클릭 한 번으로 팟캐스트 생성

업로드한 문서를 AI가 두 화자의 대화 형식으로 요약합니다. 운전 중, 산책 중에도 자료를 "들을" 수 있습니다.

#### 🔊 자연스러운 대화체

딱딱한 문서 읽기가 아닌 두 사람의 토론 형식

#### 🌍 다국어 지원

한국어 포함 다국어로 생성 가능

#### 💾 MP3 다운로드

생성된 오디오 파일을 저장 가능

#### ⚙️ 커스텀 가이드

특정 청중, 강조 내용 지정 가능

# 실전 활용 사례

이런 용도로 활용하면 생산성이 크게 향상됩니다.

🎓

### 학업 & 연구

- 논문 여러 편을 한 노트북에 모아 비교분석
- 강의 자료 업로드 후 시험 대비 Q&A
- 참고문헌 자동 정리
- 리뷰 논문 초안 작성 지원

💼

### 업무 & 보고서

- 긴 보고서 신속 요약 및 핵심 추출
- 회의록 여러 개 통합 분석
- 산업 리포트 비교 인사이트
- 발표 자료 스크립트 생성

📰

### 콘텐츠 제작

- 유튜브 영상 내용 텍스트로 정리
- 웹 아티클 수집 후 뉴스레터 작성
- 책 요약 및 독서 노트
- 팟캐스트 원고 자동 생성
