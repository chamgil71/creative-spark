companypool — 기업 마스터 DB

  • 지원: HWPX·HWP·DOCX·PPTX·XLSX·PDF·MD
  • python convert_to_md.py --input doc.hwpx --output doc.md
  • python gui/app.py — GUI 실행

filetools — 파일 관리·무결성

  • 액션: Keep / Rename / Move / Copy / Unzip / PDF_Convert
  • streamlit run frontend/app.py — 웹 UI
  • python backend/pipeline.py --manifest storage/manifest.xlsx
