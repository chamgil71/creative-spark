Introduction

클로드 코드란 무엇인가요?

설치하기 전에 클로드 코드가 어떤 도구인지 간단히 알아봐요!

  • 💬
    말로 하는 코딩 도우미 "이 코드가 왜 오류가 나요?" 처럼 우리말로 물어보면 AI가 답해줍니다.
  • 📁
    내 컴퓨터 파일과 직접 연결 인터넷 창에 붙여넣기 할 필요 없이, 내 컴퓨터 파일을 직접 분석합니다.
  • ⚡
    터미널(명령 창)에서 동작 검은 화면의 터미널에서 실행하지만, 사용 방법은 매우 간단합니다!
  • 🔄
    Claude.ai 와는 다른 도구 claude.ai(웹사이트)와 별개로 설치해서 쓰는 전문 개발자 도구입니다.

2
Prerequisites

설치 전 준비물을 확인해요

아래 항목들을 미리 준비하시면 설치가 훨씬 매끄럽게 진행됩니다.

💻

지원 운영체제

Windows, macOS(맥), Linux 모두 지원됩니다. 본인의 운영체제에 맞는 설치 방법을 선택해 주세요.

필수 확인
🟢

Node.js (버전 18 이상)

클로드 코드가 동작하려면 Node.js가 필요합니다. 아직 없다면 설치 과정에서 함께 안내해 드려요.

필수 설치
🔑

Anthropic 계정

claude.ai 계정이 있으면 됩니다. Claude Pro ($20/월) 이상 구독 또는 API 키가 필요합니다.

필수
📶

인터넷 연결

설치 중 파일 다운로드 및 인증에 인터넷이 필요합니다. 안정적인 Wi-Fi 환경을 권장합니다.

필수
⚡

관리자 권한

설치 시 컴퓨터 관리자 비밀번호를 묻는 경우가 있을 수 있습니다. 미리 비밀번호를 확인해 두세요.

경우에 따라
🐙

Git (선택)

코드 버전 관리를 위한 Git이 있으면 더 편리합니다. Windows 사용자는 Git for Windows를 권장합니다.

권장 사항
💡
Node.js가 뭔가요? JavaScript 프로그래밍 언어를 컴퓨터에서 직접 실행할 수 있게 해주는 도구입니다. 클로드 코드 내부적으로 이것을 사용하기 때문에 미리 설치되어 있어야 합니다. 마치 한글 문서를 열기 위해 '한글(HWP)' 프로그램이 필요한 것과 같아요!
3
Installation

본인의 운영체제를 선택해 주세요

사용하시는 컴퓨터 종류에 맞는 탭을 눌러주세요. 각 탭마다 맞춤 설치 방법을 안내해 드립니다.

1
터미널 열기
명령어를 입력하는 검은 화면 창을 열어요

맥에서 터미널을 여는 방법은 두 가지입니다.

방법 1 (쉬운 방법): 키보드에서 ⌘ Command + Space 를 누르고, 검색창에 터미널 또는 Terminal을 입력한 뒤 Enter를 눌러주세요.

방법 2: Finder(파인더) → 응용 프로그램 → 유틸리티 → 터미널을 클릭하세요.

💡
터미널은 컴퓨터와 텍스트로 대화하는 창입니다. 무섭게 생겼지만 명령어만 잘 입력하면 아무 문제 없어요!
2
Node.js 설치 여부 확인
이미 설치되어 있는지 먼저 확인해봐요

터미널에 아래 명령어를 입력하고 Enter를 눌러주세요.

$node --version

결과가 v18.0.0 이상이면 이미 설치되어 있으니 Step 3으로 넘어가세요!

오류가 나거나 낮은 버전이면 Node.js를 설치해야 합니다. 아래를 따라주세요.

$curl -fsSL https://claude.ai/install.sh | bash
💡
위 명령어가 무엇인가요? Anthropic 공식 설치 스크립트입니다. 이 명령어 하나로 Node.js 확인부터 클로드 코드 설치까지 자동으로 해줍니다!
3
클로드 코드 설치
실제로 설치하는 명령어를 입력합니다

터미널에 아래 명령어를 입력하세요. (방법 1이 가장 쉽습니다!)

방법 1: 공식 설치 스크립트 (가장 권장)

$curl -fsSL https://claude.ai/install.sh | bash

방법 2: npm으로 설치

$npm install -g @anthropic-ai/claude-code
⚠️
주의! sudo npm install -g ... 처럼 앞에 sudo를 붙이지 마세요. 권한 오류가 발생할 수 있습니다.

설치가 완료되면 아래 명령어로 정상 설치 여부를 확인하세요.

$claude --version
✅
버전 번호(예: 2.1.0)가 나오면 설치 성공입니다! 다음 단계로 넘어가세요.
1
PowerShell 열기
Windows의 명령 창을 열어요

Windows 키보드에서 ⊞ Windows 키를 누르고 PowerShell을 검색해서 클릭해 주세요.

💡
또는 Win + R 를 누르고 powershell을 입력해도 됩니다.
2
공식 설치 스크립트 실행
Anthropic 공식 설치 스크립트를 사용합니다

PowerShell에 아래 명령어를 입력하고 Enter를 눌러주세요.

PS>irm https://claude.ai/install.ps1 | iex
💡
irm이 뭔가요? irm은 인터넷에서 파일을 다운로드하는 명령어, iex는 그 파일을 실행하는 명령어입니다. Anthropic 공식 홈페이지에서 설치 프로그램을 받아서 바로 실행하는 것이에요.
⚠️
보안 오류가 나는 경우: PowerShell에서 실행 정책 오류가 발생하면 아래 명령어를 먼저 입력하세요.
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
3
Node.js 별도 설치 방법 (위 방법 실패 시)
공식 스크립트가 안 될 때 수동으로 설치하는 방법

① nodejs.org에 접속하여 LTS 버전을 다운로드해서 설치하세요.

② 설치 완료 후 PowerShell을 완전히 닫았다가 다시 여세요.

③ 아래 명령어로 Node.js 설치를 확인하세요.

PS>node --version

④ 버전이 확인되면 클로드 코드를 설치합니다.

PS>npm install -g @anthropic-ai/claude-code
4
설치 확인
제대로 설치되었는지 확인합니다
PS>claude --version
✅
버전 번호가 나오면 설치 성공입니다! 다음 단계(첫 번째 실행)로 넘어가세요.
💡
'claude'는 인식할 수 없는 명령이라고 나와요! PowerShell을 완전히 닫고 새로 열면 해결되는 경우가 많습니다. 그래도 안 된다면 컴퓨터를 재시작해 보세요.
1
터미널 열기
리눅스에서 터미널을 실행합니다

Ubuntu/Debian: Ctrl + Alt + T 단축키로 터미널을 열 수 있습니다.

또는 화면 좌측 상단의 애플리케이션 메뉴에서 '터미널'을 검색하세요.

2
공식 설치 스크립트 실행
macOS와 동일한 방법으로 설치합니다

터미널에 아래 명령어를 입력하세요.

$curl -fsSL https://claude.ai/install.sh | bash

설치 후 셸 설정을 새로 고침하세요.

$source ~/.bashrc

zsh를 사용 중이라면 아래를 입력하세요.

$source ~/.zshrc
3
Node.js 수동 설치 (필요 시)
Node.js가 없을 때 설치하는 방법

Ubuntu/Debian 계열:

$sudo apt update && sudo apt install nodejs npm -y

Node.js 설치 후 클로드 코드 설치:

$npm install -g @anthropic-ai/claude-code
✅
claude --version을 입력해서 버전이 나오면 성공입니다!

Troubleshooting

설치 중 문제가 생겼나요?

가장 자주 발생하는 문제와 해결 방법을 모아봤습니다. 아래 항목을 클릭하면 해결 방법이 펼쳐집니다.

설치는 됐지만 터미널이 claude 명령어 위치를 아직 모르는 경우입니다.

$source ~/.bashrc   # bash 사용자
$source ~/.zshrc   # zsh 사용자 (맥 기본)

그래도 안 된다면 터미널을 완전히 닫고 새로 여세요. Windows라면 컴퓨터를 재시작해 보세요.

npm이 시스템 폴더에 쓰려고 할 때 발생하는 권한 오류입니다. 아래처럼 npm 전역 설치 경로를 내 폴더로 바꿔주세요.

$mkdir ~/.npm-global
$npm config set prefix '~/.npm-global'
$echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.zshrc
$source ~/.zshrc

이후 다시 npm install -g @anthropic-ai/claude-code를 실행하세요.

Node.js를 최신 LTS 버전으로 업그레이드해야 합니다.

macOS: nodejs.org 에서 LTS 버전을 다운로드해서 설치하거나, Homebrew가 있다면:

$brew install node

Windows: nodejs.org에서 LTS 다운로드 후 설치 프로그램 실행

Linux (Ubuntu):

$sudo apt update && sudo apt install nodejs -y

인증을 초기화하고 다시 시도해보세요.

$claude auth logout
$claude auth login

그래도 안 된다면 claude.ai에서 Pro 또는 Max 구독이 활성화되어 있는지 확인해주세요. 무료 계정으로는 클로드 코드를 사용할 수 없습니다.

CMD나 일반 PowerShell에서 WSL(Linux) 없이 실행할 때 발생합니다. 공식 PowerShell 설치 스크립트를 사용하면 대부분 해결됩니다.

PS>irm https://claude.ai/install.ps1 | iex

그래도 문제가 있다면 claude doctor 명령어로 진단해보세요.

PS>claude doctor

인터넷 연결 상태를 먼저 확인해주세요. 클로드 코드는 매 응답마다 Anthropic 서버와 통신합니다.

회사나 학교 네트워크(방화벽)를 사용 중이라면 api.anthropic.com 접근이 막혔을 수 있습니다. IT 담당자에게 문의하거나, 개인 데이터(핫스팟)로 시도해보세요.

FAQ

자주 묻는 질문

설치 전후로 많이 물어보시는 질문들을 모았습니다.

클로드 코드는 무료인가요?
클로드 코드 자체는 무료로 설치할 수 있지만, 사용하려면 Claude Pro($20/월) 이상 구독 또는 Anthropic API 키(사용량 기반 요금)가 필요합니다. Node.js는 완전 무료입니다.
Claude.ai와 클로드 코드의 차이가 뭔가요?
Claude.ai는 웹 브라우저에서 쓰는 채팅 서비스입니다. 클로드 코드는 내 컴퓨터 파일과 직접 연결되어 동작하는 개발자용 도구입니다. 둘 다 같은 AI를 사용하지만 목적과 방식이 다릅니다.
코딩을 전혀 모르는데 써도 되나요?
네! 한국어로 자유롭게 질문하면 됩니다. 다만 설치 과정에서 터미널 명령어를 입력해야 하므로 이 가이드를 잘 따라와 주세요. 설치 후 사용은 채팅처럼 편합니다.
Mac, Windows, Linux 모두 되나요?
네, 세 운영체제 모두 공식 지원됩니다. 이 가이드의 각 탭에서 운영체제별 설치 방법을 안내하고 있습니다.
설치 후 업데이트는 어떻게 하나요?
클로드 코드는 실행할 때마다 자동으로 최신 버전을 확인하고 업데이트합니다. 수동으로 업데이트하려면 터미널에 claude update를 입력하면 됩니다.
제 코드나 파일이 외부에 유출되나요?
클로드 코드가 파일을 읽을 때 해당 내용이 Anthropic 서버로 전송됩니다. 민감한 정보(비밀번호, 개인정보 등)가 담긴 파일은 주의하여 사용하세요. 자세한 내용은 Anthropic 개인정보처리방침을 확인하세요.
삭제(제거)는 어떻게 하나요?
터미널에서 npm uninstall -g @anthropic-ai/claude-code를 입력하면 삭제됩니다. 공식 스크립트로 설치했다면 ~/.claude 또는 ~/.local/bin/claude 파일을 삭제하면 됩니다.
한국어로 대화할 수 있나요?
물론입니다! 클로드 코드는 한국어를 완벽하게 지원합니다. 질문도 한국어로, 답변도 한국어로 받을 수 있습니다.

🎉 설치 완료! 이제 시작해봐요

아래 명령어들로 클로드 코드의 다양한 기능을 탐험해보세요!

$ claude
$ claude "이 코드를 설명해줘"
$ claude doctor
