Gemini 모델 라인업

Gemini Ultra

가장 강력한 멀티모달 추론. 복잡한 과학·수학·코딩 문제에 탁월. 전문가급 성능.

Gemini Pro / Pro Vision

일상 업무와 비즈니스에 최적화. 텍스트·이미지·PDF 분석. Google Workspace 통합.

Gemini Flash

빠른 속도와 낮은 비용. API 대량 처리, 실시간 응답이 필요한 서비스에 적합.

Gemini Nano

Android 기기에서 인터넷 없이 실행. Pixel 폰, Galaxy AI 탑재. 완전한 프라이버시.

2

핵심 기능 & 강점

🌐실시간 Google 검색 연동

최신 정보를 실시간으로 검색하여 답변에 포함. 뉴스, 주가, 최신 연구도 즉시 확인 가능. 다른 AI 대비 최대 강점.

🖼️네이티브 멀티모달

텍스트·이미지·영상·음성·문서를 하나의 대화에서 처리. PDF·PPT 업로드 후 즉시 분석·요약·질의응답.

🎨Imagen 3 이미지 생성

Google의 최신 이미지 생성 모델 통합. 고품질 사진, 일러스트, 다이어그램 생성. Gemini Pro에서 직접 사용.

🎙️Deep Research & Gems

복잡한 주제를 심층 분석하는 Deep Research 기능. Gems로 특정 역할의 AI 에이전트를 만들어 재활용.

💻코드 실행 (Python)

Python 코드를 대화 중 직접 실행하고 결과 확인. 데이터 분석, 차트 생성, 계산을 실시간으로 수행.

🔗Google 생태계 완전 통합

Gmail·Drive·Docs·Sheets·YouTube 직접 접근. \"내 지난주 이메일 요약해줘\" 같은 개인화 작업 가능.

효과적인 프롬프트 & Gems 설정

Gems 시스템 지침 예시 — 업무 보고서 작성 에이전트
역할: 당신은 한국 공공기관 전문 보고서 작성 AI입니다.
능력:
- Google Drive에서 관련 문서를 검색하여 참고
- 최신 정책 동향을 Google 검색으로 확인
- 표·차트를 포함한 구조화된 보고서 작성
출력 형식:
1. 개요 (3줄 이내)
2. 현황 분석 (표 포함)
3. 시사점 및 제언
# Gems는 claude.ai의 프로젝트와 유사한 개인화 에이전트

고급 활용 — AI Studio & API

🔬 Google AI Studio 핵심 기능

프롬프트 설계 도구
시스템 지침·예시·온도 조절을 시각적으로 설정. 최적 프롬프트 실험 가능.
Grounding with Search
API 호출 시 Google 검색을 자동으로 연동. 항상 최신 정보 기반 답변 생성.
Function Calling
외부 API와 연결하여 실제 서비스를 제어하는 AI 에이전트 구축.
무료 API 티어
AI Studio에서 개인 프로젝트는 무료로 API 사용 가능. 개발 테스트에 적합.
