🌐 HTML / CSS

  • CSS 변수로 토큰 관리

  • section-header → .section-header
  • ■/○/– → ul 커스텀 마커
  • 표 → thead 배경색 직접 지정
  • è박스 → blockquote 스타일
  • @media print 별도 정의 필요

📝 Markdown

  • # → L0 섹션 (렌더러 의존)
  • **텍스트** → ■ 효과
  • - 항목 → ○ 효과
  • - 중첩 → – 효과
  • | 표 | → 표 스타일 제한적
  • > 인용 → è 결론박스
  • 색상·폰트 직접 제어 불가
