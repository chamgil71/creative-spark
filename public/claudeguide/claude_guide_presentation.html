Claude 프로젝트(Projects) 설정 내의 System Prompt 영역에 고정하여, 대화방들이 일관성 있게 행동하게 만드는 특화 프롬프트 템플릿입니다.

① 도서 요약 및 독서 모임 가이드

긴 텍스트를 구조화하고 토론 질문을 뽑아냅니다.

[역할] 독서 토론 모임의 전문 모더레이터이자 서평가. [출력 형식] 도서의 핵심 요약(주제, 타겟층, 찬반 논점)을 계층 구조로 정리하고, 참가자들이 활발히 논의할 수 있는 질문 3가지를 매번 도출할 것. [제약] 줄거리를 나열하기보다 저자의 핵심 주장과 한계점에 초점을 맞출 것.
② 비즈니스 이메일 톤 조율사

원하는 의도를 정중하고 매끄러운 격식체로 변환합니다.

[역할] 20년 경력의 대기업 비서실 소속 커뮤니케이션 전문가. [출력 형식] 사용자가 보낸 감정 섞인 문구나 조각난 지시사항을 분석하여 격식 있는 이메일 양식(인사말, 본문 핵심사항, 맺음말)으로 3가지 버전(표준형, 제안형, 협상형)을 제시할 것.
③ 개인 맞춤 건강 및 피트니스 코치

식단과 운동 이력을 분석하여 현실적인 제안을 줍니다.

[역할] 예방 의학 지식을 겸비한 전문 스포츠 영양사 및 트레이너. [출력 형식] 일일 식사 일지와 운동 강도를 분석하여 영양소 섭취 균형 비율(탄단지) 피드백과 개선이 필요한 식습관 팁 2가지를 제공할 것. [제약] 극단적인 굶기나 약물 권장 조언은 철저히 배제할 것.

실전 SKILL.md 스킬 파일 작성 예시 3종

Claude 프로젝트 컨텍스트에 SKILL.md 파일을 업로드하여 AI에게 전문적인 작업 절차와 특정 도구 사용법을 내재화시키는 팁입니다.

✍️
① 블로그 포스팅 및 카피라이팅 스킬
**목적**: 블로그 글의 가독성을 높이고 SNS 공유율을 높이는 구성 규칙.
**행동 강령**: 본문 시작 시 시선을 끄는 훅(Hook) 문장을 배치하고, 가독성을 위해 적절한 이모지를 매 소제목 옆에 배치하며, 폰 화면에서 읽기 편하도록 2~3문장마다 줄바꿈을 적용한다.
📋
② 회의록 자동 정제 및 액션 아이템 스킬
**목적**: 음성 녹음 텍스트를 깔끔한 비즈니스 회의록 서식으로 구조화하는 스킬.
**행동 강령**: 잡담은 배제하고 회의 일시, 참석자, 주요 논의 주제, 결정된 사항을 표로 정리한 뒤, 담당자별 해야 할 일(To-do)과 마감일 리스트를 하단에 명시한다.
✈️
③ 여행 일정 계획 및 동선 배치 스킬
**목적**: 낭비되는 시간과 동선을 최소화하는 일자별 여행 플래너 구성 스킬.
**행동 강령**: 오전/오후/저녁 시간대별로 실제 이동 거리와 예상 소요 시간을 표기하고, 주변 맛집과 대체지 정보를 슬라이드 카드 형식으로 레이아웃하여 제공한다.

클로드 아티팩트(Artifacts) 200% 활용법

아티팩트는 대화 흐름을 방해하지 않고 독립 패널에서 고정된 시각 결과물을 조율하는 강력한 렌더링 샌드박스입니다.

1
아티팩트 동작 원리

Claude가 한 세션 내에서 HTML, CSS, JS, SVG, Mermaid 다이어그램 등의 구조적 코드를 작성하면 대화창 우측에 전용 탭이 열리며, 로컬에 저장되거나 실시간 브라우저 실행이 가능해집니다.

2
실시간 비주얼 목업 및 시각 피드백

비개발자 사용자도 '우리 아이 학예회 초대장 페이지를 만들어줘'라고 지시하여 아티팩트 창에 뜬 예쁜 핑크색 웹 초대장을 직접 보면서 '바탕에 노란색 별 가루 애니메이션을 추가하고 글씨를 더 키워줘' 같이 시각적 피드백을 전달할 수 있습니다.

3
직관적인 결과물 소장 및 공유

아티팩트 패널 하단의 'Download' 버튼을 누르면 단 한 장의 HTML 파일로 저장되므로, 이 파일을 카카오톡이나 이메일로 친구들에게 전송하여 브라우저에서 바로 열어볼 수 있도록 전달할 수 있습니다.

2. Claude Code CLI 자율 에이전트 마스터

터미널 환경에서 스스로 파일을 읽고 쓰며 빌드 오류를 수정하고, 명령어 실행 권한을 제어하는 인텔리전트 CLI 에이전트 Claude Code의 설치 및 실무 응용법을 다룹니다.

Claude Code CLI 실행 매개변수 및 명령어 총정리

claude 명령 뒤에 플래그를 조합해 자동화 수준과 분석 경로를 지정할 수 있으며, 쉘 내부에서 슬래시 명령어 세션을 관리합니다.

기본 CLI 파라미터 (Flags)

터미널 쉘에서 실행 시 옵션

  • •claude : 인터랙티브 대화방 실행
  • •claude \"질문\" : 일회성 작업 후 종료
  • •claude -y : 모든 파일 변경/명령어 실행 승인 단계를 생략하고 자율 처리
  • •claude doctor : 네트워크 상태 및 권한 진단
  • •claude update : 최신 CLI 엔진 업그레이드
내부 슬래시 명령어 (Slash)

대화방 내부 쉘에서 활용하는 제어 명령

  • •/help : 명령어 도움말 조회
  • •/compact : 컨텍스트를 요약 정리하여 토큰 비용 절감
  • •/undo : 방금 전 단행된 로컬 소스 코드 파일 패치를 이전 상태로 롤백
  • •/settings : 에이전트 자동 승인 여부 등 설정 창 로출
  • •/exit : 대화 종료 및 시스템 복귀
MCP(Model Context Protocol) 관리

외부 도구를 연동하기 위한 설정 명령

  • •claude mcp add [name] : 신규 도구 추가
  • •claude mcp remove [name] : 연동 해제
  • •claude mcp list : 현재 등록된 MCP 리스트 조회

클로드 루틴(Claude Routine) 가이드

에이전트가 로컬 환경에서 작업을 안전하게 수행하도록 통제하고 빌드/테스트를 유기적으로 주도하는 표준 개발 루틴 가이드라인입니다.

1
1단계: 파일 탐색 및 구조 분석 루틴

먼저 전체 디렉토리 트리나 `package.json`, `tsconfig.json` 등을 탐색하여 개발 언어와 프레임워크 스택을 파악한 뒤 타겟 소스 파일을 엽니다.

2
2단계: 정밀 부분 패치(Diff Patch) 루틴

전체 파일을 다시 쓰지 않고, 수정이 필요한 줄(Lines)만 정확히 포인팅하여 오리지널-업데이트 블록 구성을 통해 코드 일부만 안전하게 갈아끼웁니다.

3
3단계: 빌드 및 유닛 테스트 루틴

수정된 소스가 컴파일에 실패하지 않는지 `npm run build`나 `vitest` 등의 유닛 테스트를 CLI 쉘 상에서 스스로 가동하여 통과 유무를 검증합니다.

4
4단계: 자율 디버깅 무한 루프 제어

테스트나 컴파일 중 에러 로그가 터미널에 수신되면 에이전트가 원인을 스스로 추적하여 다시 패치하고 재검증하는 피드백 루프를 완수합니다.

Claude Code에 MCP 서버를 추가 등록하여 로컬 데이터베이스를 조회하거나 실시간 웹 정보를 AI 컨텍스트에 주입하는 실제 설정 흐름입니다.

GitHub 연결을 통한 모바일 원격 제어 워크플로우

모바일 환경은 키보드 타이핑과 터미널 제어가 대단히 어렵습니다. 이러한 하드웨어 제약을 극복하기 위해 GitHub Actions와 Claude Code를 연계하여 스마트폰으로 로컬/서버 코딩 작업을 대리 기동하는 고급 아키텍처입니다.

1. 스마트폰 제어
GitHub Issue 등록 모바일 GitHub 앱으로 리포지토리에 Issue 생성
\"src/App.tsx의 로그인 버그 패치 후 빌드해줘\"
2. GitHub Actions 트리거
CI/CD 가동 Issue가 생성되면 워크플로우가 가동되어 클라우드 주자(Runner) 실행
터미널 상에 Claude Code 도커 컨테이너 기동
3. Claude Code 자율 해결
코드 패치 & 빌드 Issue 본문 질문을 파라미터로 입력받은 에이전트 작동
App.tsx 파일 로컬 수정 후 npm run test 수행 완료
4. 모바일 PR 승인
Pull Request 제출 Claude Code가 생성한 브랜치 및 PR 자동 업로드
모바일 폰에서 리뷰 내역 확인 후 원터치 Merge 승인 완료

3. Claude Co-work & Dispatch & Design

본 장에서는 데스크탑 앱, 크롬 브라우저를 활용한 코워크(Co-work) 방안과 복잡한 개발 요구사항을 분업하는 디스패치(Dispatch), 그리고 UI/UX를 시각화하는 디자인(Design) 노하우를 안내합니다.

Claude Co-work: 데스크탑 앱 및 크롬 브라우저 활용

인간 기획자 및 개발자와 실시간으로 화면을 조율하며 동료처럼 작동하는 환경 설정 팁입니다.

💻
데스크탑 전용 앱 (Desktop App)
**글로벌 호출 단축키**: `Alt + Space`(윈도우) 또는 `Cmd + Space`(macOS) 설정을 통해, 작업 중인 문서나 웹 브라우저 위로 Claude 창을 플로팅 창으로 즉시 소환합니다.
**파일 드래그앤드롭**: 분석하고자 하는 PDF 기획 문서나 표 데이터가 담긴 Excel 파일을 끌어다 놓기만 해도 즉시 맥락에 업로드되어 신속한 질의가 시작됩니다.
🌐
크롬 브라우저 확장도구 (Extension)
**웹 페이지 즉각 요약**: 현재 브라우징 중인 뉴미디어 기사나 전문 지식 블로그 글을 복사해 붙여넣을 필요 없이 확장도구 단축 버튼 클릭으로 핵심 3줄 요약이 완료됩니다.
**라이브 정보 탐색**: 웹 서핑 도중 난해한 해외 용어나 개념을 발견하면 마우스 우클릭 메뉴의 'Claude로 분석'을 통해 사이드바 창에서 깊이 있는 보완 설명을 얻습니다.

Claude Dispatch: 멀티 에이전트 분업 및 오케스트레이션

하나의 대화방에 모든 역할을 요구하는 대신, 고유 성격을 부여받은 가상의 에이전트들을 기동하여 업무 성격에 맞춰 역할을 배분(Dispatch)하는 전략입니다.

데이터 수집 및 핵심 요약
① 리서치 에이전트
  • 웹 검색 및 업로드된 PDF 보고서를 심층 탐색
  • 분석 대상의 장단점 및 팩트 데이터 추출
  • 관련 출처 및 통계 수치 정합성 확인
문서 작성 및 스토리라인 조립

Claude Design: UI/UX 목업 및 컴포넌트 마크업 설계

Claude의 시각 결과물 도출 능력과 SVG 인라인 코딩 스킬을 연계하여 현대적인 반응형 디자인 시스템을 단시간에 설계하는 방법입니다.

1
사용자 친화적 가계부 플래너 화면 설계

Claude에게 HSL 컬러 스키마가 적용된 반응형 일일 가계부 지출 차트 웹 화면을 아티팩트 패널에 렌더링하게 하여, 터치 한 번으로 카테고리별 지출 금액이 갱신되는 정밀 목업을 완성합니다.

2
SVG 인라인 벡터 디자인 고속 생성

외부 이미지 서버에서 가져올 필요 없이 '모던 카페 로고로 쓸 수 있는 미니멀한 머그잔 모양의 SVG 코드를 작성해줘'라고 지시하여 크기를 조절해도 화질이 깨지지 않는 청결한 벡터 아이콘 파일을 획득합니다.

3
디자인 컴포넌트 이식 및 협업 전달

완성된 화면의 미려한 CSS 스타일 가이드와 디자인 토큰을 복사하여, 내부 기획 디자이너 및 프론트엔드 담당자에게 전달함으로써 초기 레이아웃 기획 공유 비용을 최소화합니다.
