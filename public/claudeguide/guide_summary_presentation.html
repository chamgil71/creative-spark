본 가이드는 md_src/guides/ 내 모든 도구들을 역할과 도메인에 따라 6대 그룹으로 명확히 분류하여 간략히 소개합니다.

AI 및 생산성 도구 6대 분류 체계

작업 성격에 따라 최적의 도구 조합을 구성해 보세요.

지식 획득
💬1. 대화형 & 검색 AI

정보 탐색, 질문 답변, 장문 요약 및 리서치에 특화된 AI 서비스 군단입니다.

업무 자동화
💻2. AI 코딩 & 에이전트

직접 터미널을 다루고 파일을 수정하며 자율 수행을 지원하는 코딩 에이전트 도구입니다.

1인 웹 구축
🚀3. 노코드 & 인프라

코딩 없이 서비스 화면을 뚝딱 만들고, 호스팅 인프라를 빠르게 연동 배포합니다.

반복 업무 제거
⚙️4. 자동화 & 연동 도구

이종 앱 간의 데이터를 자동 연결하고, 주기적 수동 반복 작업을 완전 차단합니다.

개인 지식창고
📝5. 스마트 생산성

노트 테이킹, 화면 고속 캡처 참조, 그리고 지식 위키를 안전하게 관리합니다.

멀티미디어 생성
🎨6. 비주얼 크리에이티브

이미지 생성, 음악 제작, 비디오 편집 등 미디어 에셋을 제작하는 창작 도구입니다.

2. [그룹 1] 대화형 & 지식/검색 AI

질문을 통해 정보를 얻거나 방대한 문서를 요약하고, 최신 트렌드를 고속 검색하는 도구들의 모음입니다.

양대 범용 LLM
Claude & ChatGPT
  • Claude: 정밀 서술, 기획서 집필, 아티팩트 독립 시각 창 특화
  • ChatGPT: 다재다능한 챗봇, 빠른 응답성, 이미지 분석 및 실시간 웹 검색
빅테크 대형 모델
Gemini & Grok
  • Gemini: 구글 오피스(Drive, Docs) 문서들과 실시간 연동 및 요약
  • Grok: 실시간 소셜 미디어(X/트위터) 트렌드 정보 즉시 분석 강점
연구 및 리서치 AI
NotebookLM & Perplexity
  • NotebookLM: 내 업무 문서, PDF를 안전하게 학습시켜 나만의 지식 비서 생성
  • Perplexity: 실시간 구글링 결과를 종합 요약 제공

3. [그룹 2] AI 코딩 및 에이전트

코드를 자동으로 생성하고, 파일을 직접 쓰며, 터미널 쉘에서 자율 명령어를 기동하여 업무 생산성을 극대화하는 코딩 어시스턴트입니다.

🤖
Claude Code CLI

터미널 쉘에 상주하며 로컬 디렉토리 내 파일을 스스로 분석하여, 소스 코드를 부분 패치하고 에러 발생 시 자율 교정 루프를 도는 CLI 에이전트.

⚡
Cursor & Copilot

VS Code 기반 차세대 AI 에디터. 인라인 코드 생성, 전체 코드베이스 컨텍스트 이해 기반 코드 분석 및 탭 완성 자동 코딩 지원.

🛠️
Antigravity & Ollama

Antigravity: 다중 에이전트 협업 오케스트레이션 플랫폼. Ollama: 내 컴퓨터에서 데이터 유출 걱정 없이 로컬 AI 모델을 가동하는 무료 오프라인 LLM.

🧠
Manus & Codex

Manus: 목표만 던지면 브라우저를 조종해 가며 자율적으로 과업을 해결하는 에이전트. Codex: API 구현 및 코딩 논리 설계 보조.

4. [그룹 3] 노코드 앱 빌더 & 풀스택 플랫폼

복잡한 서버 관리나 호스팅 설정 없이, 한마디 말과 간단한 마우스 클릭만으로 나만의 홈페이지나 사내용 웹 앱을 직접 배포하는 라인업입니다.

노코드 화면 생성 및 프로토타이핑
  • Lovable: 자연어 대화만으로 완성형 풀스택 웹 앱 빌드 및 호스팅
  • v0.dev: 텍스트로 묘사한 세련된 컴포넌트 마크업 코드를 즉석 렌더링

개발 지식이 전혀 없어도 화면 UI를 예쁘게 그리고 동작하는 웹 어플리케이션을 완성합니다.

창업 아이디어 고속 MVP 검증에 최적
인프라 및 서버리스 연결 플랫폼
  • Supabase: 클릭으로 완성하는 Postgres 데이터베이스 및 로그인 관리
  • Vercel: 프론트엔드 자동 빌드 및 글로벌 도메인 배포
  • Cloudflare: 글로벌 전송망 및 고성능 CDN 제공
  • Resend: 고객 알림 발송용 이메일 API

데이터베이스, 보안 인증, 배포 파이프라인을 복잡한 인프라 세팅 없이 즉시 바인딩합니다.

안정적인 서비스 실시간 가동 및 관리

5. [그룹 4] 자동화 & 연동 & 개발 환경

여러 서비스 앱을 유기적으로 엮어 수동으로 데이터를 복사하고 옮기는 작업을 자동화하고, 작업 환경의 효율을 끌어올리는 도구들입니다.

1
n8n (업무 자동화 워크플로우)

서로 다른 웹 서비스(Gmail, Slack, Notion 등)를 시각적으로 연결해, 새로운 메일이 오면 내용을 요약해 Slack으로 전송하는 식의 무인 자동 업무 루프를 형성합니다.

2
Playwright (브라우저 자동화 및 테스트)

마치 사람이 직접 사이트를 방문하듯 크롬 브라우저를 가동하여 매일 아침 특정 사이트의 정보를 수집(스크래핑)하거나 기능이 마비되지 않았는지 자동 점검합니다.

3
GitHub & VS Code (버전 제어 및 편집기)

팀원들과 코드를 조율하여 저장하는 클라우드 저장소 GitHub와, 다양한 AI 어시스턴트 플러그인을 장착해 코딩 속도를 높여주는 에디터인 VS Code입니다.

4
Devtools & Terminal AI (브라우저 진단 및 CLI 보조)

크롬 개발자 도구를 통해 웹 디자인 레이아웃을 현장에서 실시간 수정 진단하고, 터미널 명령이 꼬였을 때 AI의 도움을 받아 즉시 수정하는 도구 모음입니다.

6. [그룹 5] 스마트 생산성 & 개인 지식베이스

개인의 메모, 수집 문서, 화면 캡처 자료들을 체계적인 디지털 서재로 관리하여 1초 만에 필요한 정보에 도달하게 해주는 생산성 허브입니다.

지식 관리의 정수
Notion & Obsidian
  • Notion: 직관적인 표 데이터베이스, 쉬운 팀 협업 및 동기화 문서
  • Obsidian: 내 PC 폴더에 마크다운 파일로 영구 소장, 노트 간 연결망(Graph View)
화면 캡처 및 참조 핀
Snipaste & Grabit
  • Snipaste: 화면의 특정 영역을 캡처한 뒤, 모니터 화면 최상단에 투명 스티커처럼 핀으로 박아두어 참고
  • Grabit: 빠른 드래그로 텍스트를 즉시 캡처 보관
지식 공개 & 초고속 검색
Quartz & Everything
  • Quartz: Obsidian으로 정리한 로컬 노트를 깔끔한 개인 홈페이지로 무료 게시
  • Everything: 내 컴퓨터 속 백만 개 파일 중 원하는 파일을 1초 만에 검색

7. [그룹 6] 디자인 & 멀티미디어 크리에이티브 AI

아이디어를 시각화하고, 고품질 마케팅 리소스를 창조하며, 오디오와 비디오 에셋을 생성 및 믹싱하는 미디어 제작 도구입니다.

🎨
이미지 생성 및 디자인/시각화 AI
**도구 리스트**: Midjourney, DALL-E 3, Stable Diffusion, Canva 등.
**핵심 역할**: 텍스트 프롬프트를 바탕으로 실사 그래픽, UI 시각 컨셉 시트, 일러스트를 무한 생성하여 디자인 기획 단계의 소통 비용을 획기적으로 줄여줍니다.
🎬
영상 및 음악/오디오 AI & CapCut
**도구 리스트**: Suno, Udio (음악 생성), Runway, Sora (영상 생성), CapCut (스마트 영상 편집).
**핵심 역할**: 간단한 묘사로 오리지널 배경음악(BGM)과 사운드 이펙트를 합성하고, CapCut의 자동 자막 생성 및 AI 컷 편집 기술을 활용해 영상 콘텐츠 제작 장벽을 낮춥니다.

8. 실무 시너지 워크플로우 시나리오 (기획 & 리서치)

개별 툴을 따로 사용하기보다, 각 목적에 맞춰 툴을 연결할 때 기획서 집필과 지식 가공 업무의 효율이 대폭 상승합니다.

1
1단계: Perplexity를 통한 실시간 트렌드 분석

신규 사업 기획을 위해 Perplexity에 최신 해외 동향과 경쟁사 데이터를 검색하고, 신뢰할 수 있는 수치 링크와 요약본을 일차 수집합니다.

2
2단계: Notion에 원시 데이터 수집 및 스토리라인 설계

수집된 텍스트와 링크를 Notion 데이터베이스에 아카이빙하고, 마인드맵 형태로 기획서의 핵심 스토리 라인과 뼈대를 구축합니다.

3
3단계: NotebookLM에 학습시켜 나만의 리서치 팟캐스트 소환

Notion에 저장된 기획 초안 문서를 NotebookLM에 업로드한 뒤, 이를 바탕으로 'Deep Dive 오디오 브리핑(대담 녹음)' 파일을 생성해 이동 중에 청취하며 추가 개선 아이디어를 도출합니다.

9. 실무 시너지 워크플로우 시나리오 (1인 창업 & 앱 배포)

자신의 비즈니스 아이디어를 30분 만에 웹 서비스로 구현하여 실제로 작동하게 만들고 글로벌 고객에게 노출시키는 혁신 파이프라인입니다.

기획 및 디자인 설계 (Claude Web & v0.dev)
  • Claude: 서비스 비즈니스 규칙 정의
  • v0.dev: 미려한 카드 및 로그인 폼 등 컴포넌트 렌더링 코드 획득

아이디어를 구체화하고, 사용자 눈에 보일 미려한 반응형 웹 화면의 마크업 레이아웃을 생성합니다.

시각 디자인 요소를 고속 빌드
서비스 기동 및 배포 (Lovable & Vercel)
  • Lovable: 대화형으로 동작 가능한 풀스택 코드 변환
  • Supabase: 회원 정보 및 지출 내역 저장을 위한 데이터베이스 즉시 구성
  • Vercel: 최종 결과물 웹 주소 퍼블리싱

데이터베이스를 부착하고 실제로 로그인과 데이터 저장이 수행되는 작동형 앱을 배포합니다.

코딩 지식 없이 완전한 가동 상태 배포

10. 실무 시너지 워크플로우 시나리오 (업무 자동화 루틴)

매일 아침 발생하는 반복 수동 작업을 지우고, 데이터 분석부터 결과 메일 발송까지 무인으로 실행하는 스마트 오피스 시나리오입니다.

1
1단계: Playwright 웹 크롤러 가동 (아침 08:00)

Playwright 스크립트가 매일 아침 특정 사이트들을 돌아다니며 환율 정보나 원자재 시세 데이터를 자동으로 긁어옵니다.

2
2단계: n8n 워크플로우 트리거 및 데이터 정제

수집이 완료되면 n8n이 동작하여 정제되지 않은 데이터 덩어리를 Notion 데이터베이스 테이블에 안전하게 정렬해 삽입합니다.

3
3단계: Claude API 연동 일일 동향 보고서 작성

n8n이 Claude API에 해당 수치들을 입력값으로 전달하면, Claude가 어제 대비 변동폭과 원인을 분석한 간결한 3줄 보고서를 메일 본문용 텍스트로 작성합니다.

4
4단계: Resend API 메일 발송 및 Slack 알림

완성된 보고서를 Resend 이메일 API를 통해 유관 부서 팀원들에게 자동 발송하고, Slack 채널에 '오늘의 동향 보고 메일 발송 완료' 메시지를 푸시하여 전사 업무를 무인 자동화합니다.
