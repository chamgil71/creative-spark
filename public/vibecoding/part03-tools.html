06

대화형 도구 — Claude / ChatGPT / Gemini

설치 없이 브라우저에서 바로 사용. 레벨 2(바이브코딩) 단계의 핵심 도구들입니다.
🤖
Claude ⭐ 추천
Anthropic · claude.ai
긴 문서 이해·정확한 추론·한국어 처리에서 강점. 코드와 문서를 동시에 다루는 작업에서 일관성이 높습니다.
📄 긴 문서 분석🇰🇷 한국어 우수💻 코드+문서🔌 MCP 연동
💬
ChatGPT
OpenAI · chatgpt.com
이미지 생성(DALL-E) 통합, GPT Store 커스텀 GPT 생태계, o3 모델의 고급 추론 능력이 강점입니다.
🎨 이미지 생성🧮 고급 추론 o3🏪 GPT Store
✨
Gemini
Google · gemini.google.com
Google Workspace(Gmail, Docs, Sheets) 네이티브 통합, Google 검색 실시간 연동이 최대 강점입니다.
📧 Gmail 연동📊 Sheets 연동🔍 실시간 검색

07

터미널 에이전트 — Claude Code / Codex CLI

AI가 직접 파일을 읽고·수정하고·실행하는 레벨 4 에이전트 도구. 브라우저를 벗어나 터미널에서 실행합니다.
⌨️
Claude Code ⭐
Anthropic · 터미널 에이전트
프로젝트 폴더에서 실행하면 AI가 전체 코드베이스를 컨텍스트로 인식. CLAUDE.md 자동 인식, Git 커밋 자동화 지원.
# 설치
npm install -g @anthropic-ai/claude-code

# 프로젝트에서 실행
cd my-project && claude

# 자율 모드 (Git 백업 후)
claude --dangerously-skip-permissions
🧠
Codex CLI
OpenAI · 터미널 에이전트
GPT-4o 기반 터미널 에이전트. 샌드박스 격리 실행으로 안전성이 높습니다.
# 설치
npm install -g @openai/codex

# 단일 작업
codex "budget_parser.py 버그 수정해줘"
항목Claude CodeCodex CLI
기반 모델Claude 4 (Sonnet/Opus)GPT-4o
파일 접근직접 접근샌드박스 격리
한국어매우 우수우수
CLAUDE.md자동 인식별도 설정
Git 연동자동 커밋 지원수동
⚠️ 필수 원칙: 에이전트 실행 전 반드시 git add . && git commit -m "에이전트 실행 전 백업". CLAUDE.md 없이 대규모 프로젝트 작업 시 일관성 없는 코드가 생성됩니다.
08

AI 에디터 — Cursor / Windsurf

기존 코드 에디터 환경에서 벗어나지 않고 AI와 실시간 협업. VS Code 기반이라 전환 비용이 거의 없습니다.
✏️
Cursor ⭐
cursor.com · VS Code 기반
VS Code의 모든 기능 + AI가 핵심 레이어로 통합. 기존 VS Code 설정을 그대로 가져올 수 있습니다.
3가지 핵심 모드
Tab → 다음 코드 자동 예측
Ctrl+K → 인라인 수정 요청
Ctrl+L → AI 채팅 (파일 자동 컨텍스트)
🌊
Windsurf
codeium.com · 독자 에디터
Cascade 에이전트가 핵심. 처음부터 에이전트 중심으로 설계되어 멀티파일 자율 수정에 강합니다.
Cascade 에이전트 특징
• 파일 생성·수정·삭제 자율 실행
• 터미널 명령 자동 실행
• 변경사항 자동 추적 및 롤백
💡 .cursorrules 설정: 프로젝트 루트에 .cursorrules 파일을 만들어 팀 전체의 AI 코딩 규칙을 통일하세요. Git에 포함시키면 모든 팀원이 동일한 규칙을 적용합니다.
09

앱 빌더 — Lovable / Bolt / v0

코드 에디터도, 터미널도 필요 없습니다. 브라우저에서 원하는 앱을 설명하면 디자인·코드·DB·배포까지 자동 완성됩니다.
❤️
Lovable ⭐
lovable.dev
자연어로 설명하면 React + Supabase 풀스택 웹앱 생성. GitHub 연동으로 코드 완전 소유.
🗄️ Supabase 자동 연결🔐 Auth 자동🐙 GitHub 연동
⚡
Bolt
bolt.new · StackBlitz
브라우저 안의 완전한 Node.js 실행 환경(WebContainers). npm 패키지 전체 활용 가능.
🌐 브라우저 내 실행📦 npm 전체🚀 즉시 미리보기
✂️
v0
v0.dev · Vercel
UI 컴포넌트 단위 생성 특화. shadcn/ui + Tailwind 기반 고품질 컴포넌트를 즉시 생성해 기존 프로젝트에 붙여넣기.
🧩 컴포넌트 단위🎨 shadcn/ui🚀 Vercel 배포
💡 조합 패턴: Lovable(초안 생성) → GitHub → Cursor(세부 수정) → Vercel(배포). 앱 빌더는 프로토타입의 출발점으로 활용하고, 세부 로직은 AI 에디터에서 다듬으세요.

자동화 플랫폼 — n8n / Make / Zapier / Antigravity

코딩 없이 "A가 발생하면 B를 해서 C에 전달"하는 자동화 흐름을 시각적으로 설계하고 자동 실행합니다.
🔄
n8n ⭐ 보안 중시
n8n.io · 오픈소스
셀프 호스팅 가능한 오픈소스. LangChain 기반 AI 노드 내장. N100 미니PC에 설치하면 월 전기세만으로 24시간 운영.
# Docker로 설치
docker run -d --name n8n \
  -p 5678:5678 \
  -v n8n_data:/home/node/.n8n \
  n8nio/n8n
🤖
Antigravity
AI 에이전트 자동화
목표를 말하면 AI가 워크플로우를 자동 설계·실행하는 차세대 플랫폼. n8n/Make를 AI가 직접 최적화.
항목n8nMakeZapier

11

로컬 AI — Ollama / Devin / SWE-agent

API 비용 없이, 인터넷 없이, 민감 데이터 외부 전송 없이 AI를 로컬에서 실행하는 방법입니다.
🦙
Ollama
ollama.ai · 로컬 LLM
Mac/Linux/Windows에서 오픈소스 LLM을 로컬 실행. n8n AI 노드와 연동하면 API 비용 없는 자동화 구현.
ollama pull qwen2.5:14b # 한국어 우수
ollama pull gemma3:12b # Google 모델
🤖
Devin
cognition.ai · 레벨 5
완전 자율 소프트웨어 엔지니어 AI. 목표만 주면 스스로 계획·코드·테스트·배포까지 처리. (레벨 5 에이전트)
🔬
SWE-agent
Princeton NLP · 오픈소스
GitHub 이슈를 자동으로 해결하는 연구용 에이전트. 오픈소스로 자체 운영 가능.
💡 n8n + Ollama 조합: N100 미니PC에 n8n + Ollama를 함께 설치하면 월 전기세 3,000원으로 완전 로컬 AI 자동화 서버를 운영할 수 있습니다. 민감 데이터 처리에 특히 유용합니다.
