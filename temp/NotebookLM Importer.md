# 2. 설치 & 사용법

::: steps
- title: Chrome 웹스토어에서 "NotebookLM Importer" 검색 후 설치
  desc: 또는 GitHub에서 직접 설치 (오픈소스). 완전 무료.
- title: Google 계정으로 로그인 (NotebookLM과 동일 계정)
  desc: 권한: NotebookLM 접근 허용. 다른 데이터에는 접근하지 않음.
- title: 저장할 NotebookLM 노트북 선택 (드롭다운)
  desc: 여러 노트북 중 원하는 곳으로 바로 지정 가능.

  desc: URL이 자동으로 NotebookLM 소스 목록에 추가됨. 나중에 NotebookLM에서 AI 분석 가능.
