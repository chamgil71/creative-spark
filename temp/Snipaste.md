# 1. Snipaste란?

일반 캡처 도구와의 결정적 차이 — **캡처한 이미지를 화면 위에 떠 있게(항상 위에 표시)** 고정할 수 있습니다. 다른 창을 열어도 사라지지 않아 참조·비교 작업이 혁신적으로 편해집니다.

::: feature-grid
- icon: 정밀
  title: 영역 캡처
  desc: 픽셀 단위 정밀 선택. 색상 피커로 특정 픽셀 색상값(HEX) 즉시 복사.
- icon: 화면
  title: 위 고정 (Pin)
  desc: 캡처 이미지를 화면 최상단에 고정. 다른 프로그램 사용 중에도 참조 가능.
- icon: 즉시
  title: 편집·마킹
  desc: 캡처 후 바로 화살표·텍스트·형광펜·모자이크 추가. 저장 없이 클립보드 복사.
- icon: 클립보드
  title: 이미지 Pin
  desc: 클립보드에 복사된 이미지도 F3으로 화면에 바로 고정. 웹 이미지 참조에 최적.
:::

# 4. 실전 활용 시나리오

::: steps
- title: 코딩 중 문서 참조
  desc: API 문서나 에러 메시지를 캡처 → Pin → 코드 에디터 위에 고정. 탭 전환 없이 개발 가능.
- title: 디자인 비교 작업
  desc: 원본 디자인 캡처 → Pin → 새 디자인 작업 화면 옆에 고정. 색상·비율 실시간 비교.
- title: 회의 중 슬라이드 참조
  desc: 발표 화면 캡처 → Pin → 자신의 메모 앱에서 내용 정리. 화면 전환 없이 회의록 작성.
- title: 색상 코드 추출 (Alt+C)
  desc: 웹사이트나 이미지에서 마음에 드는 색상의 HEX 코드를 즉시 복사. 디자이너 필수.
:::
