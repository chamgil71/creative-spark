# 1. Codex(현재의 GPT 모델)는 어떤 도구인가

Codex는 수십억 줄의 오픈소스 코드를 학습하여 자연어(사람의 언어)를 프로그래밍 코드로 번역하는 OpenAI의 AI 모델입니다. GitHub Copilot을 구동하는 핵심 엔진으로 처음 등장했으며, 현재는 GPT-4o와 o1 모델 등으로 진화하여 단순한 자동완성을 넘어 복잡한 시스템 아키텍처 설계와 로직 추론까지 수행하는 수준에 이르렀습니다.

## 한눈에 보는 핵심 기능

::: icon-grid
- icon: 🗣️
  title: 자연어 → 코드 변환
  desc: 파이썬으로 특정 웹페이지의 제목을 크롤링하는 스크립트를 짜줘"라는 말만으로 작동하는 코드를 생성합니다.
- icon: 🔄
  title: 코드 번역기
  desc: 레거시 코드(예: VBA, PHP)를 최신 프레임워크(예: Python, React) 코드로 정확하게 변환합니다.
- icon: 🐛
  title: 디버깅 및 리뷰
  desc: 에러가 발생한 코드를 던져주면, 원인을 분석하고 해결된 코드를 즉시 제안합니다.
- icon: 📚
  title: 자동 문서화
  desc: 복잡하고 주석이 없는 코드 블록을 분석하여 이해하기 쉬운 주석과 README 파일을 작성해 줍니다.
:::

# 2. 실전 활용: API로 나만의 도구 만들기

웹 인터페이스(ChatGPT)를 넘어, API를 통해 내 프로젝트에 Codex의 능력을 심는 방법입니다.

## 기본 API 연동 흐름

::: steps
- title: API 키 발급
  desc: OpenAI 플랫폼에 가입하고 결제 수단을 등록하여 프로젝트 전용 API 키를 발급받습니다.
- title: 환경 설정
  desc: 파이썬 등 사용하는 언어에서 `openai` 공식 라이브러리를 설치합니다.
- title: 프롬프트 엔지니어링
  desc: 시스템 프롬프트에 "너는 파이썬 데이터 분석 전문가야. 오직 코드만 반환해"와 같이 역할을 부여합니다.
- title: API 호출
  desc: 사용자의 입력(자연어)을 받아 `gpt-4o` 모델로 전송하고 응답받은 코드를 출력하거나 실행합니다.
- title: 안전성 검증
  desc: AI가 생성한 코드를 자동 실행할 경우, 샌드박스(안전한 격리 환경) 내에서 실행되도록 방어 로직을 구성합니다.
:::

# 4. 마무리 체크리스트

## AI 코드 사용 시 주의사항

- [ ] 생성된 코드에 무한 루프나 메모리 누수 같은 잠재적 결함이 없는지 눈으로 한 번 더 확인했는가?
- [ ] 회사 내부의 민감한 데이터나 인증 키(API Key), 개인정보를 프롬프트에 포함하지 않았는가?
- [ ] 사용하려는 라이브러리가 최신 버전인지 확인했는가? (AI가 구버전 문법을 제시할 수 있음)

> 💡 **TIP**: AI에게 코드를 수정해달라고 할 때는 전체 코드를 다시 짜달라고 하기보다, "이 함수의 이 부분 로직만 수정해서 전체 코드로 다시 보여줘"라고 지시해야 기존 구조가 망가지는 것을 막을 수 있습니다.
