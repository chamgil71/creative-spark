# 1. 왜 Everything인가?

::: compare-2col
- title: 🐌 Windows 기본 검색
  meta: 인덱스 없는 폴더 탐색기 검색 | 드라이브 전체 스캔 | 로딩 인디케이터 대기 | 파일 하나씩 순차 검색
  desc: "소요 시간: 30초 ~ 5분"
  color: "#EF4444"
- title: ⚡ Everything
  meta: Win+Alt+E로 실행 | 파일명 타이핑 시작 | 입력과 동시에 결과 표시 | 전체 드라이브 즉시 완료
  desc: "소요 시간: 0.1초"
  color: "#22C55E"
:::

> **💡 동작 원리:** Windows NTFS의 MFT(Master File Table)를 직접 읽어 모든 파일 정보를 메모리에 인덱싱. 설치 직후 빠르게 인덱스 구축 후 실시간 업데이트.

# 2. 검색 문법 & 고급 쿼리

::: feature-grid
- tag: "*.xlsx"
  title: 특정 확장자 파일 전체 검색
  desc: "예: *.pptx, *.pdf, *.py"
- tag: "보고서 2025"
  title: 두 단어 모두 포함 (AND)
  desc: 띄어쓰기 = AND 조건
- tag: "ext:xlsx dm:today"
  title: 오늘 수정된 Excel 파일
  desc: dm:yesterday / dm:thisweek
- tag: "path:C:\Projects *.py"
  title: 특정 폴더 내에서만 검색
  desc: 경로 + 파일명 조합
- tag: "size:>10mb *.mp4"
  title: 10MB 초과 MP4 파일
  desc: "size:<1mb, size:1mb..100mb"
- tag: "regex:2025\d{4}"
  title: 정규식 패턴 검색
  desc: 고급 사용자 패턴 매칭
:::

> **⭐ 자주 쓰는 쿼리 저장:** 자주 사용하는 검색식은 Bookmarks로 저장 가능. 예: "*.pptx dm:thismonth" → "이달 PPT 파일"로 즐겨찾기.

# 4. 설치 & 설정

::: steps
- title: voidtools.com 접속 → Everything 다운로드 및 설치 (무료)
  desc: 32/64비트 선택. 설치 파일 크기 2MB 이하로 매우 가벼움.
- title: 최초 실행 시 자동 인덱싱 시작 (수십 초 ~ 수 분)
  desc: SSD 환경에서는 100만 파일도 1분 이내 완료. 이후 실시간 업데이트.
- title: Tools → Options → 글로벌 단축키 설정
  desc: Win+Alt+E 또는 원하는 단축키로 변경. 시작 프로그램에 등록 권장.
- title: 시스템 트레이에 상주하며 백그라운드 실행
  desc: CPU·RAM 점유율 거의 0%. 항상 켜두어도 성능 영향 없음.
:::

> **💡 PowerToys Run 연동:** Microsoft PowerToys 설치 후 Alt+Space → Everything 플러그인 활성화. PowerToys Run 검색창에서 Everything 검색 동시 사용 가능.
