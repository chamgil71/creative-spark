# 2. 주요 기능

::: feature-grid
- icon: 페이지
  title: 전체 이미지 수집
  desc: 현재 탭의 모든 이미지를 자동 감지. 보이지 않는 영역까지 스캔.
- icon: 크기
  title: 필터
  desc: 최소 가로·세로 픽셀 설정. 아이콘·로고 같은 소형 이미지 제외 가능.
- icon: 형식
  title: 필터
  desc: JPG·PNG·WebP·SVG 등 원하는 형식만 선택 다운로드.
- icon: 미리보기
  title: 선택
  desc: 다운로드 전 이미지 목록 미리보기. 필요한 것만 체크 후 일괄 저장.
:::

# 3. 설치 & 사용법

::: steps
- title: Chrome 웹스토어에서 "Grabify Image Downloader" 또는 "Grabit" 검색 후 설치
  desc: 유사 이름의 익스텐션 다수 존재. 별점·리뷰 확인 후 선택.
- title: 이미지가 많은 웹페이지로 이동 (Pinterest, Behance, 쇼핑몰 등)
  desc: 디자인 레퍼런스, 상품 이미지, 포트폴리오 사이트에서 특히 유용.
- title: 툴바에서 Grabit 아이콘 클릭 → 이미지 목록 자동 로드
  desc: 크기 필터로 200×200px 이상만 표시 설정 권장.
- title: 원하는 이미지 체크 또는 전체 선택 → 다운로드
  desc: ZIP 파일로 일괄 다운로드. 파일명 자동 생성.
:::
