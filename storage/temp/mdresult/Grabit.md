# 2. 주요 기능

::: icon-grid
- icon: 🖼️
  title: 페이지 전체 이미지 수집
  desc: 현재 탭의 모든 이미지를 자동 감지. 보이지 않는 영역까지 스캔.
- icon: 🔍
  title: 크기 필터
  desc: 최소 가로·세로 픽셀 설정. 아이콘·로고 같은 소형 이미지 제외 가능.
- icon: 📁
  title: 형식 필터
  desc: JPG·PNG·WebP·SVG 등 원하는 형식만 선택 다운로드.
- icon: 👁️
  title: 미리보기 선택
  desc: 다운로드 전 이미지 목록 미리보기. 필요한 것만 체크 후 일괄 저장.
:::
