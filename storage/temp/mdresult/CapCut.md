# 1. CapCut은 어떤 도구인가

# 2. 기본 편집 흐름

CapCut은 복잡한 편집 프로그램을 처음부터 배우기보다, "클립 넣기 → 자르기 → 자막 넣기 → 음악 넣기 → 내보내기" 흐름으로 익히는 편이 좋습니다.

## 첫 프로젝트 만들기

::: steps
- title: 새 프로젝트 생성
  desc: 모바일 앱 또는 데스크톱에서 새 프로젝트를 만들고 편집할 영상을 불러옵니다.
- title: 클립 자르기
  desc: 필요 없는 앞뒤 부분을 잘라내고 중요한 장면만 남깁니다.
- title: 자동 자막 생성
  desc: 음성이 있는 영상이라면 자동 자막을 생성하고 오타를 수정합니다.
- title: 음악과 효과 추가
  desc: 영상 분위기에 맞는 음악, 전환, 필터를 가볍게 추가합니다.
- title: 비율과 해상도 확인
  desc: 쇼츠나 릴스는 9:16, 유튜브 일반 영상은 16:9로 맞춥니다.
- title: 내보내기
  desc: 게시할 플랫폼에 맞춰 해상도와 프레임을 선택한 뒤 저장합니다.
:::

## 초보자가 먼저 익힐 기능

| 기능 | 어디에 쓰나 | 추천 이유 |
| --- | --- | --- |
| Split | 클립 자르기 | 가장 자주 쓰는 기본 편집 |
| Auto Captions | 자동 자막 | 숏폼 시청 유지율을 높임 |
| Text Style | 자막 디자인 | 브랜드나 채널 톤을 맞춤 |
| Remove Background | 배경 제거 | 인물 중심 쇼츠에 유용 |
| Templates | 빠른 제작 | 디자인 감각이 부족해도 결과가 좋음 |

# 3. 숏폼 제작 패턴

좋은 숏폼은 길이가 짧은 것보다 구조가 분명한 영상입니다. 처음 2초에 이유를 만들고, 중간에 핵심을 보여주고, 마지막에 행동을 유도합니다.

## 추천 구성

::: compare-grid
- title: 튜토리얼형
  desc: 문제 제시 → 단계 설명 → 결과 확인 순서로 구성합니다.
  note: 예: 30초 사용법
- title: 제품 소개형
  desc: Before → 핵심 장점 → 사용 장면 → 구매/문의 유도 흐름이 좋습니다.
  note: 예: 신제품 릴스
- title: 정보 요약형
  desc: 3가지 팁, 5가지 실수처럼 목록형으로 빠르게 전달합니다.
  note: 예: 체크리스트 쇼츠
:::

## 작업 시간을 줄이는 요령

::: feature-grid

# 4. 내보내기와 운영

플랫폼마다 잘 보이는 비율과 길이가 다릅니다. 같은 영상을 모든 플랫폼에 그대로 올리기보다, 비율과 첫 화면 문구를 조금씩 조정하는 편이 좋습니다.

## 플랫폼별 기본값

| 플랫폼 | 추천 비율 | 추천 길이 | 체크 포인트 |
| --- | --- | --- | --- |
| YouTube Shorts | 9:16 | 15-60초 | 첫 2초 문구와 자막 가독성 |
| Instagram Reels | 9:16 | 7-30초 | 음악, 썸네일, 자막 스타일 |
| TikTok | 9:16 | 10-45초 | 빠른 전개와 반복 시청 |
| 일반 YouTube | 16:9 | 자유 | 챕터, 제목, 썸네일 |

## 업로드 전 체크리스트

- 첫 화면만 보고도 주제를 알 수 있는가
- 자막이 모바일 화면에서 읽히는가
- 음악 볼륨이 목소리를 가리지 않는가
- 로고나 중요한 글자가 화면 하단 UI에 가리지 않는가
- 플랫폼에 맞는 비율로 내보냈는가

> 💡 TIP: 같은 영상이라도 첫 문장과 썸네일 문구를 바꾸면 반응이 달라집니다. CapCut 프로젝트를 복제해서 A/B 버전을 만들어두면 비교하기 쉽습니다.
