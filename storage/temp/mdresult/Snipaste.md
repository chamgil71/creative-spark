# 1. Snipaste란?

일반 캡처 도구와의 결정적 차이 — **캡처한 이미지를 화면 위에 떠 있게(항상 위에 표시)** 고정할 수 있습니다. 다른 창을 열어도 사라지지 않아 참조·비교 작업이 혁신적으로 편해집니다.

::: icon-grid
- icon: ✂️
  title: 정밀 영역 캡처
  desc: 픽셀 단위 정밀 선택. 색상 피커로 특정 픽셀 색상값(HEX) 즉시 복사.
- icon: 📌
  title: 화면 위 고정 (Pin)
  desc: 캡처 이미지를 화면 최상단에 고정. 다른 프로그램 사용 중에도 참조 가능.
- icon: 🎨
  title: 즉시 편집·마킹
  desc: 캡처 후 바로 화살표·텍스트·형광펜·모자이크 추가. 저장 없이 클립보드 복사.
- icon: 📋
  title: 클립보드 이미지 Pin
  desc: 클립보드에 복사된 이미지도 F3으로 화면에 바로 고정. 웹 이미지 참조에 최적.
:::

# 3. 핵심 단축키

F1

화면 캡처 시작

드래그로 영역 선택. ESC로 취소.

F3

클립보드 → Pin

복사된 이미지를 화면에 즉시 고정.

Ctrl+C

캡처 후 클립보드

캡처 영역 선택 후 Enter 대신 사용.

Alt+C

색상 피커

픽셀 색상 HEX 값 즉시 복사.
