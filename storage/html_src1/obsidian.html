Core Concept

옵시디언이란?

옵시디언은 마크다운(.md) 파일을 기반으로 동작하는 로컬 지식 관리 도구입니다. 모든 데이터는 내 컴퓨터에 저장됩니다.

🧠

두 번째 뇌 (Second Brain)

PKM(개인 지식 관리) 방법론을 실현하는 도구. 생각과 정보를 체계적으로 수집·연결·활용할 수 있습니다.

📁

Vault (볼트)

옵시디언의 작업 단위. 하나의 폴더가 하나의 볼트가 됩니다. 모든 노트와 첨부파일이 이 폴더 안에 저장됩니다.

🔗

내부 링크 ([[wikilink]])

[[노트이름]] 형식으로 노트 간 연결. 이 링크들이 쌓여 지식 그래프가 완성됩니다.

🏷️

태그 & 프로퍼티

#태그와 YAML 프론트매터로 노트를 분류·필터링할 수 있습니다.

🗺️

그래프 뷰 (Graph View)

모든 노트와 연결 관계를 시각화. 지식의 클러스터와 고립된 노트를 한눈에 파악할 수 있습니다.

🔌

플러그인 생태계

공식 코어 플러그인과 1,400개 이상의 커뮤니티 플러그인으로 기능을 무한히 확장할 수 있습니다.

2
Key Features

꼭 알아야 할 핵심 기능

옵시디언을 강력하게 만드는 주요 기능들을 소개합니다.

①

📎 백링크 (Backlinks)

현재 노트를 링크한 다른 노트들의 목록을 자동으로 표시합니다. "이 개념이 어디서 언급되었는가"를 역추적할 수 있어 지식의 맥락을 풍부하게 만들어줍니다.

자동 생성
②

🪄 Canvas (캔버스)

무한 화이트보드에 노트, 이미지, 웹페이지를 배치하고 연결할 수 있습니다. 아이디어 정리, 마인드맵, 프로젝트 계획에 활용됩니다.

코어 플러그인
③

📆 데일리 노트 (Daily Notes)

날짜별 노트를 자동 생성합니다. 저널링, 일일 회고, 할 일 관리 등에 활용됩니다. 템플릿을 설정하면 매일 같은 형식으로 시작할 수 있습니다.

코어 플러그인
④

🔍 전체 텍스트 검색

볼트 내 모든 노트의 내용을 즉시 검색합니다. 정규식, 태그 검색, 경로 검색 등 고급 쿼리도 지원합니다.

기본 기능
⑤

📐 템플릿 (Templates)

미리 만들어둔 노트 구조를 새 노트에 즉시 적용합니다. 회의록, 독서 노트, 프로젝트 계획 등 반복적인 형식을 자동화합니다.

코어 플러그인
⑥

☁️ 옵시디언 Sync / Publish

공식 유료 서비스. Sync는 기기 간 동기화($4/월), Publish는 노트를 웹사이트로 공개($8/월)할 수 있습니다. iCloud, Git 등으로 무료 동기화도 가능합니다.

유료 서비스

💡 그래프 뷰 활용 팁
  • 노드 크기: 링크가 많을수록 노드가 커집니다 — 허브 노트를 식별할 수 있습니다
  • 색상 필터: 태그별로 노드 색상을 다르게 설정하면 카테고리가 한눈에 보입니다
  • 고립 노트: 연결이 없는 노트를 찾아 링크를 추가하면 지식 그래프가 풍부해집니다
  • 로컬 그래프: 현재 노트와 연결된 노트만 보여주는 뷰도 활용해보세요

Plugins

필수 커뮤니티 플러그인

설정 → 커뮤니티 플러그인 → 탐색에서 설치할 수 있습니다.

📅

Templater

고급 템플릿 기능. 날짜, 파일명, 스크립트를 템플릿에 삽입할 수 있습니다.

커뮤니티
✅

Tasks

볼트 전체의 할 일을 한 곳에 모아서 관리합니다. 마감일, 우선순위 설정 가능.

커뮤니티
📊

Dataview

노트를 데이터베이스처럼 쿼리합니다. 특정 태그, 날짜, 속성을 가진 노트 목록을 동적으로 생성합니다.

커뮤니티
🤖

Copilot

ChatGPT, Claude 등 AI를 옵시디언 내에서 직접 사용합니다. 노트 요약, 번역, 질문 응답 등.

커뮤니티
🗂️

Kanban

노트 파일로 칸반 보드를 생성합니다. 프로젝트 관리와 작업 흐름 시각화에 활용됩니다.

커뮤니티
🌐

Web Clipper

브라우저에서 웹페이지를 클리핑하여 옵시디언 볼트로 저장합니다. (공식 확장)

공식
🎨

Excalidraw

옵시디언 내에서 손으로 그린 듯한 다이어그램을 작성합니다. 노트에 직접 삽입 가능.

커뮤니티
⏱️

Periodic Notes

주간, 월간, 분기별, 연간 노트를 자동으로 생성하고 관리합니다.

커뮤니티

Pro Tips

실전 활용 팁

옵시디언을 더 잘 활용하기 위한 방법론과 노하우입니다.

🗂️ PARA 방법론

Projects / Areas / Resources / Archives 4개 폴더로 모든 정보를 분류합니다. 가장 널리 사용되는 PKM 구조입니다.

🌱 Zettelkasten (제텔카스텐)

각 노트는 하나의 아이디어만 담습니다. 짧고 원자적(Atomic)인 노트를 만들고 링크로 연결하면 지식이 유기적으로 성장합니다.

🤖 AI와 함께 쓰기
  • Copilot 플러그인으로 노트를 Claude/ChatGPT와 연결하여 요약·확장·번역
  • 웹에서 찾은 정보를 Web Clipper로 저장 후 AI로 가공
  • NotebookLM에 볼트의 핵심 노트를 업로드하여 대화형 질의응답 활용
⚡ 속도 향상 팁
  • Quick Switcher로 마우스 없이 노트 간 이동
  • 커맨드 팔레트(⌘P)로 모든 기능에 키보드로 접근
  • Workspace 저장으로 자주 쓰는 패널 배치를 저장·불러오기
  • 스타 기능(즐겨찾기)으로 핵심 노트 빠르게 접근
