Canva — 누구나 디자이너

🎨
Canva
canva.com · 브라우저 기반 디자인 플랫폼
무료 / Pro $15/월
📐
수천 가지 템플릿
PPT·포스터·SNS·명함·로고·유튜브 썸네일 등 카테고리별 전문 템플릿. 선택 후 텍스트·색상만 바꾸면 완성.
🤖
AI 기능 (Magic Studio)
텍스트로 이미지 생성, 배경 제거, 사진 품질 향상, 텍스트 자동 생성. 디자인 초보도 전문가 수준 결과물.
👥
실시간 협업
링크 공유로 여러 명이 동시 편집. 댓글·버전 관리·승인 기능으로 팀 디자인 작업 효율화.
📤
다양한 출력 형식
PDF·PNG·MP4·GIF·PPT·SVG 등으로 즉시 내보내기. 인쇄 주문도 Canva 내에서 바로 가능.
📊 보고서·발표 자료 빠르게 제작 (템플릿 → 내용만 교체)
📱 SNS 콘텐츠 자동 크기 조절 (인스타·유튜브·블로그 동시 제작)
🎤 행사 홍보물·배너·현수막 제작
✂️ 이미지 배경 제거 후 투명 PNG 다운로드
💡 꿀팁: Canva에서 "Presentation" 선택 후 "AI로 프레젠테이션 생성" 클릭 → 주제 입력만 하면 자동으로 구성된 PPT 초안 완성. ChatGPT와 함께 사용하면 내용도 자동 채워짐.
F

Figma — 협업 UI/UX 디자인

🖌️
Figma
figma.com · 클라우드 기반 UI/UX 디자인 & 프로토타이핑
무료 / Pro $15/월
🔗
실시간 동시 편집
개발자·기획자·디자이너가 같은 파일에서 동시 작업. 변경사항 즉시 반영. 버전 히스토리 무제한.
📱
프로토타이핑
화면 전환 애니메이션, 클릭 인터랙션 설정. 코딩 없이 앱/웹 동작 시뮬레이션 제작.
🧩
컴포넌트 시스템
버튼·아이콘·카드를 컴포넌트로 만들어 전체 디자인 시스템 구축. 수정 시 전체 자동 반영.
💻
개발자 핸드오프
CSS 값, 여백, 색상 코드 자동 추출. 개발자가 디자인 파일에서 바로 스펙 확인 가능.
🖥️ 웹/앱 UI 디자인 및 와이어프레임 제작
🎯 클릭 가능한 프로토타입으로 사용자 테스트
🏢 팀 디자인 시스템 구축 (컬러·타이포·컴포넌트)
🤝 개발팀에 정확한 스펙 전달 (핸드오프)
💡 Figma AI (Dev Mode): 선택한 디자인 요소의 코드를 React·CSS·SwiftUI로 자동 생성. 디자이너가 직접 개발 코드를 추출할 수 있어 개발자와 소통 비용 대폭 절감.
G

Gamma — AI 프레젠테이션

⚡
Gamma
gamma.app · AI 기반 프레젠테이션·문서·웹페이지 생성
무료 / Plus $10/월
⚡
AI 즉시 생성
주제 한 줄 입력 → 10초 내 완성된 프레젠테이션 생성. 슬라이드 구성·내용·디자인 모두 자동.
🎨
아름다운 기본 디자인
전문 디자이너가 만든 테마를 AI가 자동 적용. 파워포인트 없이도 세련된 결과물.
🌐
웹 공유 & 임베드
링크만으로 공유. 방문자 수·클릭 분석. 노션·슬랙·이메일에 바로 임베드 가능.
📄
PPT/PDF 변환
완성된 Gamma 문서를 PPT·PDF로 내보내기. 기존 PPT 업로드 후 AI로 재디자인도 가능.
Gamma AI 생성 프롬프트 예시
주제: 2025년 AI 도구 활용 전략 — 공공기관 업무 혁신
청중: 비개발자 공무원
분량: 10슬라이드
톤: 전문적이면서 이해하기 쉽게
📊 보고·업무 발표 자료 10분 만에 제작
📚 교육 자료·강의 슬라이드 AI 자동 구성
🏢 제안서·기획서를 웹 페이지 형태로 공유
🔄 기존 PPT를 업로드해서 AI 리디자인
💡 Claude + Gamma 조합: Claude로 발표 내용 초안 작성 → 복사 → Gamma "텍스트 붙여넣기로 생성" 선택 → 자동 슬라이드화. 30분 만에 완성도 높은 발표 자료 완성.
