핵심 기능 & 강점

🔍
출처 인용 답변
모든 답변에 실제 웹 출처를 번호로 표시. 클릭하면 원문 확인 가능. 환각(hallucination) 최소화. 학술·리서치 작업에서 타 AI 대비 압도적 신뢰성.
📊
Deep Research
복잡한 주제에 대해 수십 개 소스를 자동 분석하여 수 페이지 분량의 종합 보고서 생성. 10~15분 소요이나 논문 수준의 리서치 결과물 제공.
🤖
다중 AI 모델 선택
Pro 플랜에서 Claude, GPT-4o, Gemini, Sonar Large 등 다양한 모델 직접 선택. 작업 특성에 맞는 최적 모델로 전환 가능.
🖼️
이미지 생성 & 분석
DALL-E·Stable Diffusion 기반 이미지 생성 내장. 이미지 업로드 후 분석·질의응답. 시각 자료와 텍스트 정보를 한 화면에서 처리.
📁
파일 & 문서 분석
PDF·Word·Excel 업로드 후 즉시 분석·요약·Q&A. 최대 수십 MB 파일 지원. 문서 내용 기반 정확한 답변과 페이지 인용 제공.
🔗
Spaces (공유 리서치)
팀원과 리서치 공간 공유. 같은 소스 기반으로 여러 명이 질문하고 결과 축적. 팀 지식베이스 역할 수행.

검색 모드 & 활용법

Quick Search
빠른 단답형 질문에 최적. 뉴스·현황·정의 등 즉각적인 정보 필요 시. 무료 사용자도 무제한.
Pro Search
복잡한 질문에 추가 검색 수행. 여러 소스 교차 검증. Pro 플랜 1일 300회 이상 제공.
Deep Research
자동 반복 검색으로 종합 보고서 생성. 시장 조사·논문 검토·경쟁 분석에 최적. Pro 필요.
