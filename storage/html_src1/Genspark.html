핵심 기능

📄
Sparkpage 생성
어떤 주제든 검색하면 AI가 수십 개 출처를 종합한 맞춤형 페이지를 즉석 생성. 공유 가능한 영구 링크 제공.
🎙️
음성·이미지 검색
텍스트 외에도 음성으로 검색하거나 이미지를 업로드하여 질의. 멀티모달 검색으로 더 자연스러운 탐색.
🤖
AI 에이전트 모드
단순 검색을 넘어 복잡한 리서치 작업을 자율적으로 수행. 여행 계획, 상품 비교, 투자 분석 등 에이전트 특화 작업.
🌍
실시간 웹 데이터
항상 최신 웹 정보 기반으로 답변. 학습 데이터 의존이 아닌 실시간 검색으로 최신성 보장.

💹
투자 리서치
특정 주식·코인의 최신 뉴스, 재무 정보, 애널리스트 의견을 종합한 Sparkpage 생성.
📚
학술 리서치
논문·학술 자료를 종합하여 특정 주제의 현재 연구 동향을 종합 정리.
🛍️
상품 비교
여러 제품의 스펙·가격·리뷰를 자동 수집하여 비교 표 자동 생성.
📰
뉴스 브리핑
