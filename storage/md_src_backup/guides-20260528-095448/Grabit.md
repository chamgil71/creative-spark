# 1. 어떤 문제를 해결하나?
::: compare-split
- title: "😫 Grabit 없이"
  desc: |
    * 이미지 우클릭 → 다른 이름으로 저장
    * 반복 × 수십 장 수동 수행
    * 일일이 수동 폴더 분류 및 정리 필요
  note: "30장 = 30분 소요"
- title: "✅ Grabit 사용 후"
  desc: |
    * 익스텐션 아이콘 원클릭 가동
    * 크기 필터 설정 (예: 200px 이상)
    * "전체 다운로드" 클릭으로 일괄 수집
  note: "30장 = 10초 완료"
:::

# 3. 설치 & 사용법
::: step-list
- title: "Chrome 웹스토어에서 Grabit 설치"
  desc: "Chrome 웹스토어에 접속하여 'Grabify Image Downloader' 또는 'Grabit'을 찾아 브라우저에 추가해 줍니다."
- title: "이미지 수집할 대상 웹페이지로 이동"
  desc: "Pinterest, Behance, Dribbble, 쇼핑몰 등 대량의 고해상도 이미지가 포함되어 있는 페이지를 열어줍니다."
- title: "익스텐션 아이콘 클릭 및 이미지 스캔"
  desc: "웹 툴바에서 Grabit 아이콘을 누르면 페이지 내의 모든 이미지 자산들이 자동으로 미리보기 그리드 목록으로 실시간 로드됩니다."
- title: "필터링 설정 후 일괄 다운로드"
  desc: "불필요한 소형 로고나 아이콘은 200×200px 이상 크기 필터를 통해 자동 거르고, 원하는 이미지만 클릭하여 ZIP 압축본으로 일괄 저장합니다."
:::

::: alert-box tip
- title: "💡 UI 레퍼런스 수집 팁:"
  desc: "Pinterest·Dribbble·Behance에서 마음에 드는 디자인 페이지를 열고 Grabit 실행 → 300px 이상 필터 적용 → 전체 레퍼런스 이미지 일괄 저장. Figma 프로젝트 시작 전 무드보드 구성에 최적."
:::
