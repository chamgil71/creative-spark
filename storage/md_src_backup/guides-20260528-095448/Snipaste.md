# 1. Snipaste란?
일반 캡처 도구와의 결정적 차이 — **캡처한 이미지를 화면 위에 떠 있게(항상 위에 표시)** 고정할 수 있습니다. 다른 창을 열어도 사라지지 않아 참조·비교 작업이 혁신적으로 편해집니다.

::: icon-grid
- icon: "✂️"
  title: "정밀 영역 캡처"
  desc: "픽셀 단위 정밀 선택. 색상 피커로 특정 픽셀 색상값(HEX) 즉시 복사."
- icon: "📌"
  title: "화면 위 고정 (Pin)"
  desc: "캡처 이미지를 화면 최상단에 고정. 다른 프로그램 사용 중에도 참조 가능."
- icon: "🎨"
  title: "즉시 편집·마킹"
  desc: "캡처 후 바로 화살표·텍스트·형광펜·모자이크 추가. 저장 없이 클립보드 복사."
- icon: "📋"
  title: "클립보드 이미지 Pin"
  desc: "클립보드에 복사된 이미지도 F3으로 화면에 바로 고정. 웹 이미지 참조에 최적."
:::

화면에 고정된 참조 창들을 다양한 방식으로 조작할 수 있습니다.

  note: "Figma 화면 캡처 | 투명도 조절 가능"
- title: "API 문서"
  desc: "파라미터 목록을 캡처해서 항상 위에 띄워두고 코딩하며 직접 타이핑 참조합니다."
  note: "파라미터 목록 | 코딩하며 참조"
- title: "회의 내용"
  desc: "중요한 슬라이드 캡처 후 항상 위에 띄워두고 요점 메모하며 흐름을 파악합니다."
  note: "슬라이드 캡처 | 메모하며 확인"
- title: "계산 결과"

:::

# 3. 핵심 단축키
이 유용한 단축키들을 활용하여 작업 효율을 극대화해 보세요.

::: columns-grid
- title: "⌨️ F1"
  desc: "화면 캡처 시작 (드래그로 영역 선택. ESC로 취소)"
  meta: "드래그 영역 선택"
- title: "⌨️ F3"
  desc: "클립보드 -> Pin (복사된 이미지를 화면에 즉시 고정)"
  meta: "화면 최상단 고정"
- title: "⌨️ Ctrl+C"
  desc: "캡처 후 클립보드 복사 (영역 선택 후 Enter 대신 사용)"
  meta: "클립보드 즉시 저장"
- title: "⌨️ Alt+C"
  desc: "색상 피커 가동 (현재 마우스 픽셀 색상 HEX 값 즉시 복사)"
  meta: "HEX 색상값 추출"
:::
